cd to the directory and type the below commands

Code:

$git remote add origin https://github.com/username/java.git /* Of course, username should be replaced with whatever your GitHub username actually is */

Explaination:

The first part is familiar; we’ve used

git add

already with files. We’ve tacked the word

origin

onto it to indicate a new place from which files will originate.

remote

is a descriptor of

origin

, to indicate the origin is not on the computer, but somewhere online.

Now type

$git push origin master
