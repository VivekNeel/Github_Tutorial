Code:

$git checkout

Explaination:

Check out the specified branch, which should have already been created with git branch. This makes the current branch, and updates the working directory to match.

Code:

$git checkout -b branchname

Explaination:

Create and check out . The -b option is a convenience flag that tells Git to run git branch before running git checkout . git checkout -b Same as the above invocation, but base the new branch off of instead of the current branch.
